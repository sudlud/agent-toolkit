---

Before writing or editing a governing doc (`AGENTS.md`/`CLAUDE.md`,`rules/*.md`, `SKILL.md`, convention docs) 
or any doc it references (directly or indirectly), run `/compact-skill-creator` up front,
not as a later cleanup, so each rule stays as compact as possible.
